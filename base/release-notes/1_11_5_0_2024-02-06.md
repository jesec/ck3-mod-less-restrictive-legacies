# 1.11.5 - Couture of the Capets

**Release Date:** 2024-02-06
**Official Announcement:** https://store.steampowered.com/news/app/1158310/view/4035856766974002822

---

Hello everybody! With Chapter III now available, we're releasing the instant unlock for the expansion pass, Couture of the Capets. We're also adding some models from our Content Creator Pack, North African Attire, as personal and court artifacts for Berber Cultures. Existing save games should be fully compatible with this update, but any mods that add or alter clothing may need compatibility patches to function properly on this version of the game.
